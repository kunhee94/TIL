### 복소수

```python
#abs함수 만들기 중에 복소수 처리가 너무 어려웠음
#x.real==복소수의 실수부 x.imag ==복소수의 허수부
# 복소수 a+bj 의 절대값==(a**2+b**2)**0.5
def my_abs(x):
    if type(x)==complex:
        return (x.real**2+x.imag**2)**0.5
    if type(x)!=complex and x>0:
        return x
    elif type(x)!=complex and x<=0:
        return -x
        
```
